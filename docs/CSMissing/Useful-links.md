# 有用的链接(工具等)


## Mac软件

- **termius**: 终端好工具, 需要自己配置一下主题和字体. 收费, 可以找破解
- **yoink**: 复制好软件. 
- **orbstack**: docker或者 linux 等轻量化替代, 感觉比 docker desktop 好用
- **bob**: 翻译, ocr 翻译等
- **upic**: 上传图片
- pdf squeezer: pdf 压缩
- Downie 4: 视频下载, 万能
- Unzip one: 解压, apple store 可下
- infuse: 播放器, 能放 HDR
- Bartender 5: 状态栏管理
- Tencent lemon
