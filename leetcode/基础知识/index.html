    数组

    • 静态数组: 顺序存储的数组, 一般在定义时就开辟一段内存, 进行增删查改时, 分情况讨论, 例如数组的末端, 中间的情况是不一样的. 另外是在增加时, 如果已经超出了内存, 那就只能重新开辟一段, 所以增加的时间复杂度是O(n).
    • 动态数组: 使用起来更加方便. python 里面只有动态数组, 这点就比较好
    • rich包可以展示出代码出错的东西, 可以配置一下
